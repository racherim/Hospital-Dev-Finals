How to run

Open SQL Server Management Studio and in the "Connect to Database Engine" window type the following:

Servername: .\SQLEXRPESS
Authentication: Windows Authentication 

Open yung Schema.sql sa database file then execute. Para mag create ng database and table locally. Next, In any order execute
the sql file; admin, doctor, patient, signun

Important execute last yung Insertions it yung mag lalagay ng data sa loob ng database automatically
